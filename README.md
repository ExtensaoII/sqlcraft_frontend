# SQLCraft

**SQLCraft** is an educational game that combines **SQL learning** with **Minecraft-inspired gameplay**.  
Players solve database challenges using SQL queries to progress through missions, unlock mechanics, and interact with the game world.

The objective of SQLCraft is to make learning SQL practical, engaging, and enjoyable by transforming database queries into in-game actions.

![SQLCraft Preview](/src/assets/preview.png)

---

## Overview

SQLCraft blends game mechanics with real SQL concepts. Instead of solving abstract exercises, players interact with a world where writing correct SQL queries directly affects gameplay and progression.

This project is ideal for:
- Beginners learning SQL
- Educational or classroom environments
- Gamified learning experiments

---

## Features

- SQL-driven missions and challenges
- Progressive difficulty curve
- Minecraft-inspired game design
- Immediate feedback on SQL queries

## Installation

This project uses node version 24. To install it, go to this [link](https://nodejs.org/en/download).

### 1. Clone the repository

```bash
git clone <project_url>
cd sqlcraft
```

### Install depedencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```

Open your browser and go to `http://localhost:5173`

## How the game works

- Each mission introduces a new objective.
- Objectives are completed by writing SQL queries
- Incorrect queries provide feedback to guide learning
